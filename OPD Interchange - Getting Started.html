GETTING STARTED


         Those  of  you who are impatient to use the OPD/QL Interchange
         System  should be able to use it successfully to copy complete
         cartridges  between  OPD  and  QL formats without reference to
         the documentation.  To achieve this do the following:


           a) Reset  (or  switch  on) your QL, put the issued cartridge
              into MDV1, and press F1 or F2.


           b) From  the  menu  that  is  displayed,  select the OPDCOPY
              program (option 1) by pressing the '2' key.


           c) Follow  the  on-screen  prompts given by this program. If
              you  are  not sure which answer to give to a prompt, then
              the  default  given  is  normally  a sensible one (merely
              press ENTER to accept it).


         For  more  sophisticated  use  you  will  have  to  read   the
         documentation.   Please take note of the points given in these
         issue notes under 'Printing or Viewing the Documentation'



         ISSUE FILES


         The  following files should be on the cartridge containing the
         OPD/QL Interchange system software:


         BOOT                Basic Program to provide a Boot up Menu.


         OPDCOPY             An  'EXECable'  program  for  easy copying
                             of  OPD  and  QL cartridges.  Also used to
                             make  backup copies of the OPD Interchange
                             System software.


         OPDLOAD             'EXECable' programs  for loading or

         OPDEND              ending the OPD Device Driver manually.


         OPDDRIVER           The code for the OPD Device Driver.


         OPDBASIC            A  Basic  program  to help with converting
                             Basic  programs  between  the  OPD and QL.
                             See  OPD/QL  Basic  Conversion  Guide  for
                             details.


         OPDCONFIG           A  Basic  program  for  configuring    the
                             OPD/QL  Interchange  software.  See OPD/QL
                             User Guide for details on how to use it.


         OPD1_DOC            Quill files containing the OPD/QL User

         OPD2_DOC            Guide.


         OPDBAS1_DOC         Quill files containing the OPD/QL Basic

         OPDBAS2_DOC         Conversion Guide.

         PRINTING or VIEWING DOCUMENTATION FILES


         When  the  software  is  supplied  on  Microdrive,  there   is
         insufficient  free  space  to allow Quill to open a work file.
         On a QL without extra memory you must therefore:


           EITHER  Print  the  document  without loading into Quill (by
                   giving the filename in the Quill PRINT command).


           OR      Copy  the  document  file  you wish to examine to an
                   empty cartridge before trying to Load it into Quill.


         The  documentation  files  have  been  laid  out  so that page
         breaks  occur  at  convenient  points if you use 12 inch paper
         (72  lines).    An allowance of 1 inch (6 lines) has been made
         for  the  top  and  bottom  margins.  If you use paper that is
         shorter  than  12  inches,  use the DESIGN command in Quill to
         change  the  Page  length,  and  the Top and Bottom margins as
         appropriate to keep the page breaks at the same point.




         KNOWN RESTRICTIONS  (in Release 2.6)


         1)  The  OPD/QL  Interchange  software  will  not  work     in
             conjunction  with  Miracle's  Gold  Card.    The Gold Card
             unfortunately  upsets  critical timing loops in the OPD/QL
             Interchange software.


         2)  The  OPD/QL  Device  Driver  will  not work in conjunction
            with  the  Eidersoft  ICE  system  or  with TASKMASTER from
            Sector  Software.    Any  attempt to use OPD/QL Interchange
            with  these  packages  causes  the QL to 'lock' which means
            that the QL has to be reset.


         3)  It  has  been  found that the option to format a cartridge
            (in  OPD  format)  on the QL is not always reliable.  There
            appears  to  be  about  a  5% chance of a READ failure when
            reading  such  a  cartridge  on the OPD.  This problem does
            not  occur if the cartridges in OPD format are formatted on
            a real OPD, and not on the QL. 

          

         4)  The  OPDCOPY  program will not perform Data Translation on
            text strings under the following conditions:

          

             - When they occur as part of an Archive Database file (one
              whose file name ends in _dbf or .dbf).


               -  When  they  occur as part of an Abacus Save file (one
              whose file name ends in _aba or .aba).


             - When they occur as part of an Easel Save file (one whose
              file name ends in _grf or .grf).

            convert  satisfactorily  except  for  the  fact  that   the

         5. MISCELLANEOUS NOTES



         Error Handling


         The  QL  Error  Handling  facilities were only introduced with
         the  JS  ROM,  and  are not present in earlier versions.  They
         are NOT documented in the Basic section of the QL User Guide.

         These  options  should  be  used with caution as they have not
         been fully validated.


         The  OPD  TRACE  statement is not documented in early versions
         of  the  OPD  Basic  manual.   The WHEN ERROR statement is not
         documented in any version of the OPD Basic Manual.

         Colour


         QL colour parameter can consist of up to 3 parameters:

                   basic colour

                   second colour

         OPD Colour can only be have a basic colour. Commands that set
